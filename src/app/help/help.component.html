The Product Vision

For Temple University students WHO wants to feel secure. TU Map Safe is a Website application THAT provides access to Temple police and gives real time GPS that support features which sends the locations to the police. Unlike the current alerts notification system OUR product provides a GUI that makes clear to the user their distance to the threat so that they can react accordingly.
